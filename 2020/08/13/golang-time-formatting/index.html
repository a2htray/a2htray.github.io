Golang -摘记- 时间格式化

Posted on 2020-08-13

时间格式处理。Copy from public resources.

>> Golang uses time package to get timestamp and date formatting

For example, in PHP, we use date(‘Ymd H:i:s’, time()) to output the time “2019-04-30 14:43:26”, such as Java. “new SimpleDateFormat(“yyyy-MM-dd HH:mm:ss”)”. However, in the Go language, “Ymd H:i:s”, “yyyy-MM-dd HH:mm:ss” is a specific number “2006-01-02 15:04:05” is the creation time of the Go language. And must be the exact number of these.

总结：需要指明具体日期进行格式化。

>> Time Formatting in Golang

预设的一些格式字符串，如下：

FORMAT EXAMPLE
ANSIC “Mon Jan _2 15:04:05 2006”
UnixDate “Mon Jan _2 15:04:05 MST 2006”
RubyDate “Mon Jan 02 15:04:05 -0700 2006”
RFC822 “02 Jan 06 15:04 MST”
RFC822Z “02 Jan 06 15:04 -0700”
RFC850 “Monday, 02-Jan-06 15:04:05 MST”
RFC1123 “Mon, 02 Jan 2006 15:04:05 MST”
RFC1123Z “Mon, 02 Jan 2006 15:04:05 -0700”
RFC3339 “2006-01-02T15:04:05Z07:00”
RFC3339Nano “2006-01-02T15:04:05.999999999Z07:00”

总结：系统提供默认的格式字符串供选择。
