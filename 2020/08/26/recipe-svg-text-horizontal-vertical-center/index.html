Recipe -SVG- text 水平垂直居中

Posted on 2020-08-26

text 水平垂直居中演示。

静态

如果父元素是 svg，使用 x="50%" y="50%"。同时：

  • text-anchor="middle" 表示水平居中
  • dominant-baseline="middle" 表示垂直居中

动态

  1. text-anchor="middle" dominant-baseline="middle" 这两个属性一定要有
  2. 通过 api 得到 <rect/> 元素的 x,y,width,height，分别为变量 rectX,rectY,width,height
  3. 设置 <text /> 的 x,y，其中
    1. x = ${rectX + width/2}
    2. y = ${rectY + height/2}

  1. How to place and center text in an SVG rectangle
