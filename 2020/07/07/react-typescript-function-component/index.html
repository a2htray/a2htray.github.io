Posted on 2020-07-07 Edited on 2020-08-14

创建一个空的 React-TypeScript 项目

基本用法

  1. 使用 interface 声明一个 props 类型，用于指定组件属性传值的类型
    1. 如 title: string，指明 <List /> 必须传一个字符串类型的 title 值
  2. React.FC<Props> 为 List 的类型，表示为一个函数组件，其属性 props 需要实现 Props 接口

使用 state

  1. useState 方法返回一个数组，使用数组解析 []，将数组中的值赋值给左边的变量
  2. items 的类型为 Array<Object>
  3. 使用 Array.map 方法，读取数组中的项并返回 []JSX.Element
  1. 通过对文本框 onChange 事件监控name 和 price 两个值
  1. 使用 ...items 解析，并对 items 重新赋值
