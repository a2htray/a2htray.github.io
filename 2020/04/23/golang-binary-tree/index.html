二叉树前序、中序、后序、层序遍历 Go 实现

主体结构

前序

顺序：根 -> 左 -> 右

在非递归形式中，利用栈的性质：

  1. 取栈顶，并输出
  2. 有右结点，入栈
  3. 有左结点，出栈
  4. 重复 1，直至栈空

中序

顺序：左 -> 根 -> 右

在非递归形式中，利用栈的性质：

  1. 根入栈
  2. 有左结点，入栈，重复 2 至无左结点
  3. 取栈顶
  4. 有右结点，入栈，重复 1

后序

顺序：左 -> 右 -> 根
