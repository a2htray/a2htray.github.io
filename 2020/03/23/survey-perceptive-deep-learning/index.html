综述笔记-深度学习框架及其应用

Posted on 2020-03-23 Edited on 2020-05-13

近期需要学习掌握神经网络的有关知识，所以找了一篇引用较高的综述，对文中信息进行摘录，以备后期回顾。

Deep learning approaches have also been found to be suitable for big data analysis with successful applications to computer vision, pattern recognition, speech recognition, natural language processing, and recommendation systems.

深度学习的应用范围：大数据分析、计算机视觉、模式识别、语音识别、自然语言处理、推荐系统。

With machine learning techniques, computers are endowed with the capability of acting without being explicitly programmed, constructing algorithms that can learn from data, and
making data-driven decisions or predictions.

在机器学习的助力下，计算机已逐渐不需要显式编程、设计算法，被赋予从数据中学会思考的能力，作出基于数据的决策与预测。

when it comes to the human information processing mechanisms (e.g. speech and vision), the performance of traditional machine learning techniques are far from satisfactory.

尽管机器学习已在许多领域发挥着巨大作用，但在对于人类信息处理机制等问题，如语音、视觉，传统机器学习的表现力远远达不到我们的预期。

In contrast to traditional machine learning and artificial intelligence approaches, the deep learning technologies have recently been progressing massively with successful applications to speech recognition, natural language processing (NLP), information retrieval, compute vision, and image analysis.

相较于传统机器学习和人工智能的方法，目前的深度学习技术已经取得巨大的进步，在语音识别、自然语言处理、信息检索、计算机视觉和图像分析等领域有着大量成功的应用。

often gets trapped in local optima.

神经网络中的反向传播策略是基于局部的梯度信息进行处理，并且初始的权重值是随机的，这会导致算法经常会陷入局部最优。

Furthermore, if the size of the training data is not big enough, NNs will face the problem of over-fitting.

在神经网络中，如果训练的数据量不够会导致出现过度拟合的现象。
