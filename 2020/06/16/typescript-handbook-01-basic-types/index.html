TypeScript 基本类型

Posted on 2020-06-16 Edited on 2020-08-14

在修改一个前端项目的过程中需要去读懂 TypeScript 的代码，所以选取了 MS 官方的教程进行学习并记录。这一系列的不在于传授经验，所以其中的部分翻译都是直接取自官方文档，希望能理解 TypeScript 的大致脉络。

介绍

主要介绍 TypeScript 中的类型以及如何声明相应类型的变量

Boolean

布尔类型，必须是 true/false

Number

数值类型，在 TypeScript 中，所有数值类型都是浮点数，并支持多种进制的表示字面量形式

String

字符串。支持双引号""与单引号''。

数组，声明数组时可指定元素的类型

Tuple

元组用于声明固定长度的数组，元素类型可以不同

越界下标会报错

Enum

枚举类型。使用更友好的方式表示数值类型的数据

默认情况下，起始值为 0。在声明时，显式声明起始值。

也可以直接设置所有的值

Any

如果不确定具体类型时，可以使用 any类型

Void

常用于函数返回类型的声明

如果变量的类型为 void，那该变量只能赋值为 null (如果没有指定 --strickNullChecks) 和 undefined。

Null 和 Undefined

在 TypeScript 中，Null 和 Undefined 也作为两种类型。和 Void 类型差不多，一般很少使用

Never

用于函数返回，any 类型的变量不能声明为 never

Object

object表示非基础数据类型，变量不能赋值类型为number、string、boolean、bigint、symbol、null 和undefined。
