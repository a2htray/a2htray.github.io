Ollama：部署与使用

发表于： 2026-03-18 分类于： 人工智能/大语言模型
字数： 1387 阅读：≈ 3分钟 浏览：

Ollama 是大模型的运行平台，本文主要介绍 Ollama 的安装与使用。

安装

听说 Ollama 类似于 Docker 一样，出了 Desktop 版本，所以这次我选择下载 .dmg 文件方式安装。

访问官网下载页面 https://ollama.com/download

下载 .dmg 安装文件

双击安装，拖动图标至 Applications

打开程序，如下图：

Desktop

Terminal

1
2

$ ollama -v
ollama version is 0.17.4

ollama 的子命令规则与 docker 相似。

使用

查看当前环境的模型

1
2
3
4

$ ollama list
NAME          ID              SIZE      MODIFIED          
qwen3.5:9b    6488c96fa5fa    6.6 GB    12 minutes ago       
qwen3.5:2b    324d162be6ca    2.7 GB    About an hour ago

启动 Ollama

  1. 点击桌面图标启动
  2. 命令行启动
1

$ nohup ollama serve &

运行模型

1
2

# 进入交互式问答
$ ollama run qwen3.5:2b

按 Ctrl + D 退出。

查看已运行模型

1
2
3

$ ollama ps  
NAME          ID              SIZE      PROCESSOR    CONTEXT    UNTIL              
qwen3.5:2b    324d162be6ca    4.1 GB    100% GPU     4096       3 minutes from now

停止运行模型

1

$ ollama stop qwen3.5:2b

!注意！

API 文档中有两个功能相似的接口：

  1. POST /api/generate
  2. POST /api/chat

两者的区别在于：/api/generate 用于单次文本的生成，/api/chat 用于多轮对话，如下两个调用示例：
