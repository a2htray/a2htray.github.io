PostgreSQL 数据类型

Posted on 2021-02-10

PostgreSQL 数据类型包含：

  • Boolean 布尔类型
  • Character 字符串类型，如 char，varchar、text
  • Numeric 数值类型，如整型、浮点类型
  • Monetary 货币类型
  • 时间类型，如 date、time、timestamp 和 interval
  • UUID
  • Array 数组类型
  • JSON
  • hstore，用于保存键值对
  • 特殊类型，如网络地址、地理数据

Boolean

Boolean 类型的可能值包括：true、false 或 null。使用关键字 boolean 或 bool 定义。

在插入语句中：

  • 关键字 true、字符串 true、字符串 yes、字符串 t、字符串 y、字符串 1 可以作为 真 true
  • 关键字 false、字符串 false、字符串 no、字符串 f、字符串 n、字符串 0 可以作为 假 false

在查询显示时，t 表示真，f 表示假。

空字符串表示 null。

Character 字符串类型

  • char(n) 定长的字符串类型，当插入的字符串小于 n 字符时会补空，大于 n 个字节时报错
  • varchar(n) 非定长的字符串类型，最多能存储 n 个字符的字符串
  • text 非定长的字符串类型，理论上可以存储无限长度的字符串

可以看出，尽管 sex 定义了 char(6)，但通过 length 方法会返回去除后置空格后的字符串长度，如 length(' abc ')=4。

Numeric 数值

PostgreSQL 支持两种不同的数值类型：

  • 整型
  • 浮点型

Integer 整型

  • smallint：有符号整数，占 2 个字节
  • int：整数，占 4 个字节
  • bigint：占 8 个字节
  • serial：自增的整数，无需显示插入

浮点型

  • float(n) 浮点数，n 表示精确度，占 8 个字节
  • real 或 float8 浮点数，占 4 个字节
  • numeric 或 numeric(p, s)，其中 p 为数值个数，s 表示小数点后几位

Monetary 货币类型

占 8 个字节。

时间类型

  • date 日期类型，存储年月日信息，占 4 个字节
  • time 时间信息，不存储年月日信息，占 8 个字节
  • timestamp 时间戳，存储年月日时分秒信息，占 8 个字节
  • interval 时间间隔，占 16 个字节

特殊类型

关键字 字节 说明 表示
