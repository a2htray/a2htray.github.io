Sass 学习

Posted on 2019-04-23 Edited on 2020-08-23

Sass (Syntactically Awesome StyleSheets)

工作中有需要用到 Sass，前端代码不好写，能写但写得不好，希望能模块化编写样式，所以需要了解一些 CSS 预处理器相关的知识。

做点储备。也许以后还会学点其他预处理器语法，但希望用不到，用好一个，到哪都任意跑。

安装

使用 npm，全局安装即可，使用淘宝的源，安装速度有提升

命令行介绍

完成安装后，在命令行下，输入 sass 查看工具的使用说明，感觉比较重要的选项有:

  • --[no-]source-map 是否生成 source map 文件
  • --watch 监听文件改动，及时编译
  • --[no-]stop-on-error 有错误发生时，立即停止

选项还有比较多，会使用 sass --watch source.scss destination.css 即可，再说前端开发框架中也会集成 sass 预处理器，很少有自己输样式编译指令的机会。
