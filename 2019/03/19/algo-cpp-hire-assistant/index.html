算法练习 -CPP- 雇用费用问题

Posted on 2019-03-19 Edited on 2020-08-23

通过程序设计的方法模拟用人单位在雇用员工过程中所需要的成本问题。

注重考虑人工成本，而不是考虑算法的时间复杂度。

问题描述

假设用人单位招聘某一岗位的员工，每面试一个人的成本是 PER_COST，每次招聘到更合适的人选，则将前一名员工辞退，辞退一名员工的成本是 PER_HIRE_COST。在数据规模在 n 的情况下，考虑用人单位的总成本。

总成本需要考虑一组员工数据中其 value 值的分布有所不同，则对于不同的数组，得到的总成本必然不同。

  • PER_COST 为常量 10
  • PER_HIRE_COST 为常量 100
  • 面试的人的能力以 value 表示，越大越好
