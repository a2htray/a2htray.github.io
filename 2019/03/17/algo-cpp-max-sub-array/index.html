最大子数组问题

问题描述

有一连续时间段(按天)、同一股票的价格，每天都在波动。使用一个数组保存，如:

需要进行一次买入、卖出操作，为获得最大利益(低买高卖)，该如何进行操作。

显然最低价出现在第8天，为63，最高价出现在第2天，为113，不可能在第8天买入，再在第2天卖出。卖出的时间必然是在买入时间之后。

暴力求解

思路: 求出任意两天之间的差值，取最大差值，同时将这两天分别记为买入时间和卖出时(需要区分前后顺序)，时间复杂度为 O(n^2)

最大子数组

思路: 我们的目的是为了求解某两天的最大差值，换一个角度，记录相邻两天的变化值，将连续的变化值进行相加，即可求得这两于的差值。取连续的变化值和的最大值即可。将问题转换成在一个数组中求解最大了数组。显然数组要比原来的数组少一个元素。

分治策略

数组 A[low, high] (以 0 起始) 分成两个尽可能数量相等的子数组 A[low, mid]、A[mid+1, high]，则最大子数组 A[i, j] 必然满足下列之一:

  • low <= i <= j <= mid
  • mid < i <= j <= high
  • low <= i <= mid < j <= high

头两种情况依然可以理解求解最大子数组，只不够在更小的规模中，而第三种情况只要从 mid 开始向左、向右计算最大值即可，看方法 findMaxCrossSubArray，时间复杂度为 O(n)。取三种情况的最大值即可。
