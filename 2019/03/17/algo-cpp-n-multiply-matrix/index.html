n*n 矩阵相乘 Strassen 算法

问题描述

矩阵相乘，即 A[n,n] * B[n,n] = C[n,n]

通用算法

矩阵 C 中的 n^2 个元素分别为 n 个数值相加而得。利用三个 for 循环即可，时间复杂度为 O(n^3)

运行后输出

Strassen 算法

利用分治策略，将矩阵相乘操作理解成各个块相乘，每一个块均为一个方阵，当为 1*1 方阵时，相乘即可。

块方阵的表示从原矩阵的两个维度(一组行下标，一组列下标)进行表示，下标使用链表进行表示，并指定链表的长度，如结构体 IndexLink。
