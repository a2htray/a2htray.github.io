监视系统活动的各种命令 df、du、watch、ps

统计磁盘使用情况

磁盘管理和问题排除的各种办法

df 和 du 是Linux中用于统计磁盘使用情况的重要命令，df 是 disk free 的缩写，du 是 disk usage 的缩写

  • -a：获取某个目录的磁盘占用，不指定 -a 只显示子目录的磁盘占用
  • -h：显示人类可读大小，如K、M
  • -c：显示总的大小，加一个total
  • -s：只输出合计信息

使用特定的单位显示

选项 说明
-b 以字节为单位显示
-k 以KB为单位显示
-m 以MB为单位显示
-B 指定一个块大小，并以块为单位
  • --exclude：排除特定文件，不进行统计
  • --exclude-form：指定文件列表文件，对该文件中记录的文件不进行统计

例子

du 提供磁盘使用情况，df 提供磁盘可用情况

计算命令的执行时间 time

格式:time COMMAND

与当前登录用户、启动日志及启动故障的相关信息

who、w、users、uptime、last、lastb

  • uptime：查看系统启动运行了多长时间
  • last：获取前一次启动及用户登录会话的信息
  • lastb：获取失败的用户登录信息

打印出10条最常使用的命令

命令的使用记录保存在~/.bash_history文件中，通过对该文件操作取最常使用的前10条

列出一个小时内占用cpu最多的10个进程 ps

ps 命令用于收集系统中进程的详细信息

工作原理：使用 sleep 命令对一段时间内CPU的使用情况进行采样，将采样信息输出到临时文件，对临时文件进行处理操作，完成过滤功能

如一个小时，以每分钟为间隔进行一次取样，以追加的方式写入临时文件，再对临时文件进行处理即可

用 watch 监视命令输出

watch 命令：在某段时间内以固定的间隔时间不断监视某个命令的输出

  • -n：指定间隔的时间
