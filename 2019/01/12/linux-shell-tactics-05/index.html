Linux -Linux Shell 脚本攻略- 05 没这么事

Posted on 2019-01-12 Edited on 2020-08-23

Linux命令行下的一些网页操作

了解 wget、curl，及如何发送 POST 请求

wget

wget 是一个用于文件下载的命令行工具

选项：

  • -t：指定下载重试次数
  • -c：接续下载
  • -Q：设置最大额，超过则停止下载

wget 有一个选项可以使其像爬虫一样以递归的方式收集网页上的所有URL链接，并逐个下载，使用 --mirror

以格式化纯文本形式下载网页

Lynx 是一个基于命令行的Web浏览器，可以将 Lynx 的纯文本格式化输出作为网页来获取

cURL入门

cURL 支持HTTP、HTTPS、FTP在内的众多协议，所下载的数据被写入 stdout

如果不希望显示进度信息，使用--silent

选项

  • -C：直接接续下载
  • -o：指定保存的文件

和 wget 不同，cURL 包含更高级的下载恢复特性，能够从特定的文件偏移处继续下载，通过指定一个偏移量来下载部分文件

  • --referer 指定参照页字符串，即从什么页面进行访问的
  • --cookie 指定 cookie，key=value，以分号;相隔
  • --cookie-jar，将 cookie 下载到具体文件
  • --user-agent：设置用户代理
  • -H：设置 Header 信息
  • --limit-rate：限速操作
  • --max-filesize：指定最大下载量，如果文件超出限额，返回非0
  • -u 完成 HTTP 或 FTP 认证，如-u username:password
  • -I 或 -head：只显示HTTP头部信息，而无须下载远程文件
  • -d 或 --data：以POST的形式发送请求
